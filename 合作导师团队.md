# 合作导师团队

整理日期：2026-02-11  
说明：本页为网页文字版，统一展示合作导师信息；头像以 Markdown 图片链接形式保留（即保留原图片链接）。

我们与多所高校与科研机构的导师开展合作，形成覆盖**大规模强化学习、后训练与对齐、优化理论、以及大模型相关方法**的联合指导网络。部分导师承担**日常指导**职责，持续参与课题推进；同时也与来自高校/研究机构的导师保持**联合培养 / 学术合作**，支持学生科研训练与学术交流。
